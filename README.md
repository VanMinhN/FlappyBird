# FlappyBird
Clone of the famous mobile game Flappy Bird created using Unity3D and C#

Game Mode: Dodge the pillar through out the level and score point when you get pass each pillar. The game goes on infinitely until you either collide with the pillar or you exit the game. 

Pc: to play game go to build/Windows/x86/FlappyBird.exe
tap left mouse to fly
